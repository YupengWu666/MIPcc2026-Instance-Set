# PaNGEA Instances

This repository contains 50 compressed MPS instances from the PaNGEA project.

## Contents

- `instances/instance_01.mps.gz` to `instances/instance_50.mps.gz`
- Format: gzipped MPS files

## Download

Clone the repository:

```bash
git clone https://github.com/YupengWu666/MIPcc2026-Instance-Set.git
```

Or download a single instance from the GitHub web interface.

## Notes

The source files were copied from `PaNGEA/data/instances/`. Local metadata files such as `.DS_Store` are intentionally excluded.
